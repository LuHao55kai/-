## 状态

没有真正的上传功能，成功选择文件后只会显示预先设好的图片（《星月夜》.jpg） 点击识别也只能显示预先设好的结果（《星月夜》梵高）

## 日志

- 优化了窗口自适应效果。
- 为选择文件添加了文件类型限制。
- 为防止用户重复发送识别请求，现在点击“识别”按钮后，该按钮将被禁用，直到识别完成显示结果。
- 添加了新的视觉特效，以表明正在识别图片，并添加了一个测试按钮，以模拟识别成功显示结果。
